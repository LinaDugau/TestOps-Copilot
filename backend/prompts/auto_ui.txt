AAA Template (обязательная секция)
ВСЕ ТЕСТЫ ОБЯЗАНЫ СТРОГО СЛЕДОВАТЬ СХЕМЕ AAA (Arrange → Act → Assert):
Каждый тест должен состоять ЧЕТКО из трех групп шагов:
1) ARRANGE — подготовка
   - инициализация данных
   - открытие страницы/экрана
   - создание предусловий

2) ACT — действие
   - пользовательские действия
   - вызов API
   - выполнение основного шага

3) ASSERT — проверка результата
   - assert statements
   - проверки UI
   - проверки API-статусов и тела

import allure
import pytest
from allure import step as allure_step
from playwright.sync_api import Page, expect
@allure.feature("Калькулятор цен Cloud.ru")
@allure.story("Начальная страница")
@allure.title("Открытие калькулятора и проверка отображения нулевой стоимости")
def test_calculator_opens_with_zero_price(page: Page):
    with allure_step("Arrange: Открыть калькулятор"):
        page.goto("https://cloud.ru/price-calculator")
    with allure_step("Act: Дождаться загрузки и получить элементы цены"):
        price_text = page.locator("text=0 ₽ в месяц")
    with allure_step("Assert: Проверить, что цена равна 0 ₽ и сделать скриншот"):
        expect(price_text).to_be_visible()
        page.screenshot(path="screenshots/zero_price.png")
        allure.attach.file("screenshots/zero_price.png", name="Нулевая стоимость", attachment_type=allure.attachment_type.PNG)
