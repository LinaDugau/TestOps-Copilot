AAA Template (обязательная секция)
ВСЕ ТЕСТЫ ОБЯЗАНЫ СТРОГО СЛЕДОВАТЬ СХЕМЕ AAA (Arrange → Act → Assert):
Каждый тест должен состоять ЧЕТКО из трех групп шагов:
1) ARRANGE — подготовка
   - инициализация данных
   - открытие страницы/экрана
   - создание предусловий

2) ACT — действие
   - пользовательские действия
   - вызов API
   - выполнение основного шага

3) ASSERT — проверка результата
   - assert statements
   - проверки UI
   - проверки API-статусов и тела
